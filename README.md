### 느낀 점
- 단순 모델은 구조적으로 한계가 있다
- 데이터 범위를 벗어나면 예측 불가
- 모델 복잡도는 반드시 제어해야 한다

> KNN은 데이터 범위를 벗어나면 예측 불가<br>모델 복잡도는 k 값에 따라 조절됨<br>단순 모델의 한계를 경험하는 단계
