# ChatGPT를 이용한 머신러닝 실습 복습용
이 브랜치는 ChatGPT를 활용하여<br>
머신러닝 개념을 복습하고 직접 구현해보는 실습 예제 모음입니다.<br>
<br>
이론을 읽는 데서 끝나지 않고,<br>
코드를 다시 작성하고 실험하며 개념을 정확히 이해하는 것을 목표로 합니다.<br>

단순 코드 실행이 아니라,<br>
✔ 개념 설명<br>
✔ 코드 작성<br>
✔ 실험 비교<br>
✔ 오류 수정<br>
과정을 반복하며 학습 내용을 체화하는 것을 목표로 합니다.

# 브랜치 방향성
이 브랜치는 프로젝트 결과물을 위한 공간이 아니라,<br>
개념을 정확히 이해하기 위한 실습 중심 학습 공간입니다.
