# 구조

```
데이터
→ 신경망 구조
→ 활성화 함수
→ 손실 함수
→ 최적화
→ 학습
→ 평가
```

# TensorFlow란?
`TensorFlow = 딥러닝 모델을 만들고 학습시키는 라이브러리`<br>

예
```
가중치 계산
행렬 곱
활성화 함수
Loss 계산
Gradient 계산
```
예를 들어 이런 계산이 있습니다.
`z = wx + b`
그리고 이것을 수천 번 반복합니다.
```
forward
backpropagation
gradient update
```
이 계산을 직접 구현하면 엄청 복잡합니다.
그래서 등장한 것이 `TensorFlow`입니다.

# 인공신경망 기본 구조
신경망은 기본적으로 *선형 모델 + 활성화 함수* 구조

```
입력
 → 선형결합(wx+b)
 → 활성화함수
 → 출력
```

# 신경망 층 구조
인공신경망은 **레이어(layer)**로 구성
```
입력층
 ↓
은닉층
 ↓
출력층
```

예 <br>
`784 → 100 → 10`

의미 <br>
```
784 : 입력 특징
100 : 은닉층 뉴런
10  : 클래스 개수
```

# Dense 레이어
`Dense(units=100, activation='relu')`

| 요소         | 의미     |
| ---------- | ------ |
| Dense      | 완전 연결층 |
| units      | 뉴런 개수  |
| activation | 활성화 함수 |

# Flatten
이미지 데이터에서 자주 등장

예 <br>
`28 x 28 이미지`

Flatten <br>
`784`

즉<br>
`2D → 1D`

예 코드 <br>
`Flatten()`

# 활성화 함수 (Activation Function)
활성화 함수는 신경망의 핵심

**역할** : 비선형성 추가
Dense만 있으면 선현모텔이 되기때문

**ReLU** :음수는 0, 양수는 그대로 | 학습이 빠르고 기울기 소실이 적음
☞ 은닉층 기본

**Sigmoid** : 그래프는 S곡선, 범위는 0~1, 이진분

**Softmax** : 다중분류에서 사용 , 활률 합 = 1

# Loss (손실 함수)
신경망은 **Loss를 최소화하는 방향으로 학습**
```
Loss ↓
→ 모델 성능 ↑
```

**Binary Cross Entropy** : 이진분류 
**Categorical Cross Entropy** : 다중분류

# Optimizer
가중치를 업데이트하는 방법
```
SGD
Adam
```

**SGD**
```
Loss 기울기 계산
→ 조금씩 이동
```

**Adam**
SGD 개선 버전
```
학습 빠름
안정적
```

# 학습과정
`model.fit(X_train, y_train)`
과정
```
1 forward
2 loss 계산
3 gradient 계산
4 weight 업데이트
```
이 과정을 
`epoch`
만큼 반복한다.

# 평가
`model.evaluate()`
역할
` 모델 성능 확인`
