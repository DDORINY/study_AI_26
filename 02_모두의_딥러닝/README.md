# 모두의 딥러닝 학습 기록
딥러닝 기초 개념을 이해하고 실제 코드 실습을 통해
모델의 동작 원리를 학습하는 개인 공부 기록 저장소입니다.

이 저장소는 **『모두의 딥러닝』 책을 기반으로 딥러닝을 학습하며 정리한 기록 공간입니다.**

단순히 코드를 실행하는 것이 아니라
딥러닝의 **개념 이해 → 코드 실습 → 결과 분석 → 응용**의 과정을 통해
모델이 **왜 이렇게 동작하는지 이해하는 것**을 목표로 공부합니다.

---

## 학습 목표

* 딥러닝 기본 개념 이해
* 신경망 구조와 동작 원리 학습
* TensorFlow / Keras 기반 모델 구현
* 모델 학습 과정과 결과 해석 능력 향상
* 실습을 통한 딥러닝 활용 능력 강화

---

## 학습 방식

이 레포지토리는 다음 방식으로 학습을 진행합니다.

1. 책의 내용을 기반으로 개념 정리
2. 실습 코드 직접 실행 및 수정
3. 결과 시각화 및 모델 동작 분석
4. 추가 실습 예제 제작 및 응용

---

## 학습 내용

* 퍼셉트론
* 인공신경망 (ANN)
* 활성화 함수
* 오차 역전파 (Backpropagation)
* 딥러닝 모델 구조
* TensorFlow / Keras 실습
* 데이터 기반 모델 학습

---

## 기록 방식

각 노트북에는 다음 내용을 기록합니다.

* 핵심 개념 정리
* 코드 설명
* 실험 결과
* 느낀 점 및 추가 학습

---

## 목적

딥러닝을 **단순 사용 수준이 아니라
모델의 원리를 이해하고 설명할 수 있는 수준까지 학습하는 것**을 목표로 합니다.
